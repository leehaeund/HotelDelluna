# HotelDelluna
 
### 호텔 예약 사이트 Spring Project

 www.parkjaeho.xyz 

## How to use  
 **회원당 하나의 예약만 가능합니다.**
1. 로그인 후 사용을 해야합니다.   비회원일 경우 예약이 ~~불가능~~ 합니다
2. 날짜를 선택 후 원하는 룸을 선택합니다.
3. 결제를 합니다.
4. 결제 후 예약 조회/취소를 클릭합니다.
5. 호텔에 대해 궁금하시다면 공지사항을 클릭합니다.


 ## Development environment
